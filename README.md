# Chocolate Shipments Data Analysis | Python, Pandas

## Project Overview
Performed initial Exploratory Data Analysis (EDA) on chocolate shipment data using Python. This script loads the dataset and generates summary statistics to understand data structure, distributions, and missing values before visualizing insights in Power BI.

## Objective
1. Load and validate chocolate shipment dataset using Pandas
2. Generate descriptive statistics to understand sales, boxes, and pricing distributions  
3. Check data quality by identifying null values and column data types
4. Prepare cleaned data foundation for advanced visualization in Power BI

## Tech Stack
- Language: Python
- Libraries: Pandas, NumPy
- Output: Console-based EDA summary for data profiling


##  Power BI Dashboard Plan
After this Python EDA, the following visuals will be created in Power BI:

1. YoY Comparison - Line Charts: Total Amount Generated Last Year vs Present Year, Boxes Sold Last Year vs Present Year
2. Geographic Analysis - Pie Chart: Amount Generated by Countries  
3. Distribution Analysis - Clustered Column Chart: Shipment Distribution by Category/Region
4. KPI Cards: Major metrics - Total Amount, Total Boxes Shipped, Total Profit
5. Product Performance - Treemap: Top Products by Revenue
6. Sales Team - Table Chart: Top 5 Salespersons by Performance
